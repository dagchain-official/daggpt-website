# AI Model Icons

Store the official SVG icons for each AI image model in this folder.

## Required Icon Files

Please save the SVG files with these exact names:

1. **flux-kontext-pro.svg** - Flux Kontext Pro icon
2. **flux-2.svg** - Flux 2 Pro icon
3. **nano-banana-pro.svg** - Nano Banana Pro / Google Gemini icon
4. **grok-imagine.svg** - Grok Imagine / xAI icon
5. **4o-image.svg** - OpenAI GPT-4o icon
6. **seedream-v4.svg** - Seedream v4 / ByteDance icon
7. **ideogram-v3.svg** - Ideogram v3 icon
8. **midjourney.svg** - Midjourney icon

## Icon Specifications

- **Format:** SVG
- **Recommended size:** 24x24px viewBox
- **Color:** Can be full color or monochrome (we'll handle sizing in code)
- **File naming:** Use kebab-case (lowercase with hyphens)

## Usage

Once you add the SVG files here, I'll update the `EnhancedImageChatInput.js` component to import and use them instead of the inline SVG code.
